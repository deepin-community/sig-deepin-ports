## 支持设备

- SG2044 EVB

## 下载并解压镜像

在[镜像列表](https://deepin-community.github.io/sig-deepin-ports/images/riscv64)中下载最新的 `.tar.xz` 格式的镜像包至本地，随后使用以下命令解压镜像与校验码至当前文件夹：

``` shell
tar -xvf <你下载的镜像名称>.tar.xz
```

解压得到

- `deepin-sg2044-riscv64-25-desktop-installer.sha256sum`
  sha256 校验和
- `deepin-sg2044-riscv64-25-desktop-installer.md5sum`
  md5 校验和
- `deepin-sg2044-riscv64-25-desktop-installer.root.ext4`
  root 镜像

校验镜像：

``` shell
# 两者一致即可
$ cat deepin-sg2044-riscv64-25-desktop-installer.sha256sum
<sha256sum>  deepin-sg2044-riscv64-25-desktop-installer.root.ext4
$ sha256sum deepin-sg2044-riscv64-25-desktop-installer.img
<sha256sum>  deepin-sg2044-riscv64-25-desktop-installer.root.ext4
```

## 刷写操作系统至 NVMe 硬盘

使用分区工具创建一个空白的 gpt 分区表，新增一个分区，此处例子中 NVMe 对应分区名为`/dev/sda1`。使用 `dd` 命令将镜像刷入对应分区。

``` shell
sudo dd if=deepin-sg2044-riscv64-25-desktop-installer.img of=/dev/sda1 status=progress
```

## 刷写 bootloader

前往[revyos linuxboot-sg204x 仓库](https://github.com/revyos/linuxboot-sg204x/actions)，进入最新的 `linuxboot-sg204x` 项，在 Artifacts 中选择最新的形如 `sg2044-bootloader-single-sg2044-upstream-*` 的项目下载并解压，得到以 `.img` 结尾的形如 `firmware_sg2044-single-sg2044-upstream-*.img` 的 bootloader 镜像文件。

### 刷写 bootloader 至 SD 卡

使用 `dd` 命令将 bootloader 写入到 SD 卡。

``` shell
sudo dd if=firmware_sg2044-single-sg2044-upstream-*.img of=/dev/sda status=progress
```

### 刷写 bootloader 至 mtd

> 注意：如果需要刷写 bootloader 至 mtd，需要先进入系统，如果不能进入系统，请先刷写 bootloader 到 SD 卡。

## 启动

将 NVMe 硬盘安装至设备，如需使用 SD 卡启动，将 SD 卡插入主板对应的 SD 卡槽，连接好 HDMI 视频输出，通电，等待设备启动至安装器界面，完成安装即可

## 注意事项

- 镜像启动时间可能较长，如需查看串口，请参见对应设备的[测试报告](https://deepin-community.github.io/sig-deepin-ports/docs)。
